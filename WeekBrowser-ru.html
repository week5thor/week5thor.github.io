WeekBrowser

Лёгкий Android-браузер с открытым кодом

WeekBrowser — простой, быстрый и приватный браузер для Android. Поддерживает JavaScript-команды, мультиоконный режим, блокировку рекламы и полностью открытый код (The Unlicense).

⚙️ Системные требования

  • Минимально: Android 5.0+
  • Рекомендовано: Android 8.0+
  • Языки: украинский, английский, русский

🌟 Основные возможности

  • Суперменю: долгое нажатие на любой элемент страницы открывает дополнительные действия — блокировка рекламы, скачивание, просмотр кода или открытие в новой вкладке.
  • Мультиоконный режим: открывайте несколько сайтов одновременно или создавайте «мультиссылки» для быстрого открытия набора вкладок.
  • Блокировщик рекламы: использует регулярные выражения для экономии памяти и блокирует видеорекламу, если возможно.
  • Менеджер позиции: сохраняйте место на странице для дальнейшего чтения.
  • Скины: более 20 тем оформления, включая динамические цвета для Android 12+.
  • Исключения для сайтов: индивидуальные настройки для каждого домена — цвета, скрипты, функции.
  • Поиск и замена URL: автоматическая перенаправка сайтов, например youtube.com → ssyoutube.com для загрузки видео.

🔧 Расширения

Поддерживается собственный формат расширений на JavaScript. Они выполняются при загрузке страницы, могут перехватывать запросы, импортироваться и экспортироваться через специальные ссылки.

⚠️ Известные проблемы

  • Загрузка: BLOB-файлы могут не работать (XHR error).
  • Интерфейс: возможна неинтуитивность — ваши отзывы приветствуются.
